üîç Troubleshooting

Can't see the Coupons menu?
  • Make sure DigiSales plugin is activated (Plugins ‚Üí Installed Plugins)
  • Verify you're logged in as an Administrator
  • Clear your browser cache and refresh (Ctrl+F5 or Cmd+Shift+R)
  • Check WordPress version is 5.8 or higher
Coupon not working on frontend?
  • Verify coupon code spelling (case-sensitive)
  • Check if current date is within the valid period
  • Ensure usage limit hasn't been reached
  • Confirm product is in the applicable products list
